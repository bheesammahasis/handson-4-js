    The second line of the input contains N integers C1,C2,C3,C4,...,CN color of 
    shirts (separated by space).
    Output Format
    Return a single integer M denoting the unique colored shirts left in the 
    wardrobe. 
    Constraints
    1<=N<=10^3
    1<=C[i]<=10^3
    Example
    Input
    6
    3 2 4 1 2 3
    Output
    2

    Input: C=3, 2, 4, 1, 2, 3
    Output: 2
